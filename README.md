# ⚡ Energy Consumption Prediction App

This project predicts energy usage (Usage_kWh) in the steel industry using Machine Learning.

## 🔹 Model
- Linear Regression

## 🔹 Features Used
- Reactive Power
- Power Factor
- CO2 Emission
- Time (NSM)
- Week Status
- Day of Week
- Load Type

## 🚀 Run Locally
```bash
pip install -r requirements.txt
python -m streamlit run app.py
